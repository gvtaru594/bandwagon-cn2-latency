## CN2 是什么，为什么它和普通线路延迟差这么多

搬瓦工的线路粗略分三档：163 普通线路、CN2 GT、CN2 GIA。

163 就是中国电信最基础的国际出口，全程走公网，晚高峰经常拥堵，延迟抖动是常事。CN2 GT（Global Transit）是 CN2 线路的入门版，回国段会经过 CN2 节点，但国际出口那一段走的还是普通线路，拥堵时有影响。CN2 GIA 全称 Global Internet Access，全程走 CN2 节点，从美国数据中心一路到你家附近的 CN2 出口，不绕路，不走公网拥堵段。

简单说：同一时段同一城市，CN2 GIA 的延迟稳定性远超 CN2 GT，更别提 163 了。

搬瓦工目前的主力 CN2 GIA 套餐分两大阵营：

- **E-Commerce 系列（CN2 GIA-E）**：美国洛杉矶 DC6、DC9 为代表，带宽从 2.5Gbps 起步，可切换 14+ 机房，$169.99/年起。
